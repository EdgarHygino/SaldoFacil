# SaldoFacil

## Rodando o Projeto

Antes de iniciar o projeto, certifique-se de ter criado o banco de dados e aplicado as migrações. Para fazer isso, siga estas etapas:

1. Abra o terminal na pasta raiz do projeto.
2. Execute o comando `dotnet ef migrations add FirstMigration -o Context/Migrations` para criar a primeira migração.
3. Execute o comando `dotnet ef database update` para aplicar as migrações e criar o banco de dados.
4. Certifique-se de que o banco de dados foi criado corretamente e está disponível.

Depois de configurar o banco de dados, você pode iniciar o projeto seguindo estas etapas:

1. Abra o projeto no visual Studio 2022.
